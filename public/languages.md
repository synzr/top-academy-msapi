# Ресурс доступных языков локализации

## Метод `GET`

|----------------|------------|-----------------------------|-----------------|
| [].name_mystat | `string`   | Полный код ISO локализации  | `"ru_RU"`       |
| [].short_name  | `string`   | Краткий код ISO локализации | `"ru"`          |
